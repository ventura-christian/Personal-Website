# My Personal Landing Page

Built with HTML and CSS. No JavaScript, no build step.

[Live Demo -> CLICK ME](https://ventura-christian.github.io/Personal-Website/)

## What This Is

A six-page static personal site:

1. **Home** (`index.html`): name, intro, and links into the rest of the site
2. **About** (`about.html`): bio blurbs, background info cards, and contact links
3. **Projects** (`projects.html`): a shuffleable sticker wall for browsing the three projects below
4. **InfraScope** (`infrascope.html`): project detail page
5. **SleeveSanctum** (`sleevesanctum.html`): project detail page
6. **Vectris** (`vectris.html`): project detail page

Every page shares one stylesheet, `css/style.css`.

## Design Direction

Dark background, graffiti and sticker visuals, hand drawn characters and scribble SVGs, organic blurb shapes, accent colors the change with the page. Retrofuturism inspired by print poster design.

## Tech Stack

- HTML5 (semantic, WCAG-checked contrast and focus states)
- Custom CSS only: variables for theming, `clamp()` for fluid type, and radio/checkbox hack patterns for the shuffle carousel and mobile nav toggle that ensures no JavaScript anywhere on the site
- Responsive across mobile, tablet, and desktop breakpoints
